Lucide direction

Use a consistent line icon style, equal stroke weight, and the same icon size grid so the launcher feels coherent rather than decorative.

Icon-first, not icon-only

Icons can dominate visually, but labels or tooltips are still needed so actions remain understandable and accessible.

Keep the set short

The launcher works best when only the most important repeated actions are shown. Everything else should route into a dedicated flow.

Current direction: Variant A is the LernHive baseline. Variant B is an optional layer for high-frequency action contexts, but it should not replace the compact base launcher.
