# local_lernhive_audience — User Documentation

## User value
Audience helps administrators and product owners group users using familiar Moodle structures and simple rule logic.

## What the user should experience
- simple, clear labels
- minimal steps
- no hidden complexity
- terminology that also works in product communication

## Release 1 experience
- Audience feels like a guided extension of Moodle, not a replacement system
- the user can work with static and dynamic Audiences
- the user can define dynamic Audiences with:
  - time-based rules
  - activity-based rules
  - profile-based rules
- rules can be combined with AND / OR logic
- create and edit flows stay simple and list-based
- terminology stays English-first and stable

## Main user journeys
- open the Audience area from the relevant LernHive entry point
- create a static Audience from known Moodle structures
- create a dynamic Audience using a small number of clear rules
- review members and rules in a simple list-based view
- understand why a user belongs to an Audience

## Release 2 note
- Circle UX is a later refinement, not part of the Release 1 baseline
- richer visual rule composition may be explored later without changing the Release 1 simplicity goal
