# local_lernhive_configuration — Master Document

**Plugin type:** local plugin
**Release target:** R1

## Purpose

Stores flavour overrides and configuration history for transparency and partner collaboration.

## Role in LernHive

This plugin is part of the LernHive ecosystem and should fit the core rules:
- Moodle stays the core
- English-first terminology
- use Moodle core strings wherever possible
- no business logic in the theme
- must stay understandable for UX, docs, marketing and sales

## Main dependencies
local_lernhive_flavour

## Main features
- override storage
- change history
- configuration comparison
- customer-visible configuration summary
