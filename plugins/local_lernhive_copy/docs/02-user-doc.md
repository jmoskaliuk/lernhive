# local_lernhive_copy - User Documentation

## User value

Create a new course quickly by reusing an existing course as a base, without
manually rebuilding structure and activities.

## Current user experience (0.3.0)

- One clear entry point from ContentHub
- Guided Simple flow for course copy with explicit fields
- Expert mode toggle for advanced copy options
- Copy runs in the background using Moodle core
- Immediate redirect to Moodle copy progress page after submit
- Template entry shows Library-backed template choices
- Last-used target category is remembered for the user

## Main user journeys

1. Open ContentHub and choose **Copy**
2. Select source course and define target course details
3. Keep participant/progress copy off for a clean cohort copy (default)
4. Submit and follow progress in Moodle copy progress UI

## Template-path journey (two-step)

1. Open ContentHub and choose **Template**
2. Pick one available template from the catalog list
3. Continue with **Simple** or **Expert** mode for that template source
4. Submit and follow copy progress in Moodle

## UX expectations

- simple labels and minimal steps
- no hidden side effects
- terminology that is consistent across product docs and UI
- mobile-friendly layout and touch-safe form controls
