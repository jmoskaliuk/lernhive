# local_lernhive_discovery — User Documentation

## User value
Explore gives learners and participants a simple way to find relevant Courses, Snacks, and Communities in the LXP Flavour.

## What the user should experience
- simple, clear labels
- minimal steps
- no hidden complexity
- terminology that also works in product communication

## Release 1 experience
- Explore is available only in the LXP Flavour
- Explore replaces Dashboard only in the LXP Flavour
- the page stays slim and easy to scan
- the user sees only four main feed blocks:
  - New in your community
  - New in content you follow
  - Popular snacks
  - Explore
- cards can show Follow and Bookmark separately
- Follow uses a star icon on cards
- Bookmark is an explicit save-for-later action

## Ranking explanation
- users should understand why something appears
- community and Audience relevance should feel strongest
- recent activity from the last 7 days matters
- Snacks may surface earlier than longer content
- likes are not part of the ranking story

## Main user journeys
- open Explore from the left navigation in the LXP Flavour
- scan a small number of clear feed blocks
- follow a relevant item directly from its card using the star action
- bookmark an item explicitly for later without changing Follow state
- understand why a result is shown when the UI surfaces the reason

## Release 2 note
- stronger personalization may improve Explore later
- Release 2 changes must keep the Explore experience understandable
