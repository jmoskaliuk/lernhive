# local_lernhive_flavour — Master Document

**Plugin type:** local plugin
**Release target:** R1

## Purpose

Loads flavour defaults and maps a system to a best-guess starting profile.

## Role in LernHive

This plugin is part of the LernHive ecosystem and should fit the core rules:
- Moodle stays the core
- English-first terminology
- use Moodle core strings wherever possible
- no business logic in the theme
- must stay understandable for UX, docs, marketing and sales

## Main dependencies
local_lernhive

## Main features
- flavour selection
- default settings load
- plugin bundle profile
- flavour-aware navigation/text defaults
