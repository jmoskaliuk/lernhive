# local_lernhive_flavour — Features

## Feature summary
Loads flavour defaults and maps a system to a best-guess starting profile.

## Planned feature set
- flavour selection
- default settings load
- plugin bundle profile
- flavour-aware navigation/text defaults

## Acceptance direction
- feature behaviour should be explicit and understandable
- strings should reuse Moodle core when possible
- UX should stay simple and mobile-friendly
- flavour-specific wording should be used only if really necessary

## Release note
Target release: R1
