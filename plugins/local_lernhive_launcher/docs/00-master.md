# local_lernhive_launcher — Master Document

**Plugin type:** local plugin
**Release target:** R1

## Purpose

Global action launcher for create/manage/configure actions.

## Role in LernHive

This plugin is part of the LernHive ecosystem and should fit the core rules:
- Moodle stays the core
- English-first terminology
- use Moodle core strings wherever possible
- no business logic in the theme
- must stay understandable for UX, docs, marketing and sales

## Main dependencies
local_lernhive

## Main features
- role-aware launcher entries
- global quick actions
- entry to ContentHub
- entry to Library
- entry to Reports
- entry to snack/community creation
