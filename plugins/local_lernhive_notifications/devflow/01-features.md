# local_lernhive_notifications — Features

## Purpose
Extend Moodle notifications for LXP needs without replacing Moodle messaging.

## Reuse from Moodle
- providers
- channels
- admin defaults
- user preferences

## LernHive extension
- daily digest
- weekly digest
- bundling of LXP events

## Default decisions for release 1
- Community updates → digest
- Follow updates → digest

## Relevant event examples
- new snack in my community
- new member in my community
- new discussion in followed content
- meaningful update in followed content

## Non-goals
- no likes-based noise
- no standalone replacement of Moodle notification architecture
