# local_lernhive_notifications — Master Document

**Plugin type:** local plugin
**Release target:** R1

## Purpose

Digest and LXP notification extension over Moodle notifications.

## Role in LernHive

This plugin is part of the LernHive ecosystem and should fit the core rules:
- Moodle stays the core
- English-first terminology
- use Moodle core strings wherever possible
- no business logic in the theme
- must stay understandable for UX, docs, marketing and sales

## Main dependencies
local_lernhive_follow, local_lernhive_discovery

## Main features
- reuse Moodle providers
- daily digest
- weekly digest
- LXP-specific event bundling
