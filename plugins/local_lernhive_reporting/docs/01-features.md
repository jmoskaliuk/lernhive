# local_lernhive_reporting — Features

## Purpose
Offer a simple reporting dashboard on top of Moodle reports and analytics.

## Release 1 report tiles
1. How many users are in my course?
2. Which courses are the most popular?
3. Course completion report

## UX principle
- dashboard with tiles
- each tile shows a key number
- click opens drilldown
- build on existing Moodle reporting where possible

## Release 2 ideas
- inactive users
- content usage trends
- community activity insights
