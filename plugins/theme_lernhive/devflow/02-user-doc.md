# theme_lernhive — User Documentation

## User value
The theme makes LernHive feel calmer, clearer, and easier to navigate without changing Moodle core behavior.

## What the user should experience
- simple, clear labels
- minimal steps
- no hidden complexity
- terminology that also works in product communication

## Release 1 experience
- navigation feels left-oriented and predictable
- top navigation shows only a few important actions
- Launcher feels like a quick action surface, not a second menu system
- cards, lists, and forms feel calmer and easier to scan than standard Moodle
- the same visual language works across School and the optional LXP Flavour

## Main user journeys
- open a page and understand the main structure immediately
- move between areas through a stable left navigation
- use the Launcher for creation and admin actions without losing orientation
- read cards, tiles, and helper surfaces without visual overload
- use the same interface comfortably on desktop and smaller screens

## Mockup goal
- the mockup should already show how Moodle pages, LernHive cards, and action surfaces fit into one coherent theme system
