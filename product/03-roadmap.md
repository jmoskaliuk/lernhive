# Roadmap

## Release 1 — Foundations

### Core
- Theme foundation
- Level system
- Launcher
- Context Helper
- Flavour setup
- Configuration history

### Content
- ContentHub
- Copy wizard
- Library import with managed content files
- minimal version visibility

### LXP
- Explore replaces Dashboard in LXP
- Feed with fixed blocks:
  - New in your community
  - New from content you follow
  - Popular snacks
  - Explore more
- Follow and bookmark
- Audience rules
- Digest notifications

### Reporting
- three primary report tiles:
  - user count in course
  - most popular courses
  - course completion

## Release 2 — Depth
- stronger personalization
- learning system / recommendation ideas
- library update UX
- improved Circle UX for audiences
- content lifecycle and version update workflow
- skills UX based on Moodle competencies and learning plans
- richer reporting dashboard
- flavour refinement for Higher Education and Corporate Academy

## Release 3+
- recommendation engine
- adaptive discovery
- deeper analytics
- content lifecycle automation
